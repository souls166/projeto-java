Gestor de Finanças
-

Esse projeto tem a finalidade de controle financeiro pessoal, onde ele ajuda a ter controle,noção e responsabilidade financeira do dia a dia.
-
Onde ele pode ajudar a identificar gastos prejudiciais, ter equilibro financeiro e até mesmo planejamento

O que ele tem?
-
controle de despesas: onde você registra as entradas(salário,vendas, cobranças,etc) e saída(compras,faturas,lazer,etc)
calculo de quanto sobra o orçamento
visualização grafica(grafico de pizza)

como rodar o projeto?
-
1.instale/clone a pasta do github

2.abra o terminal

3.crie a pasta 

4.coloque todos os codigos em arquivos separados dentro do nano ou apenas arraste para a pasta

5.rode "nano Main.java"

6.rode "javac .*java"

7.e por ultimo, java"GestordeFinancas"
